Welcome to the GoofyOS Info Page

Introduction

Welcome to GoofyOS! This application allows you to create, manage, and run custom HTML apps within the GoofyOS desktop environment. Learn more about its features below.

Core Features

1. Create Custom Apps
You can create your own apps by providing a name and HTML content. Once created, these apps will appear on the sidebar for easy access. Apps can include any HTML, CSS, and JavaScript to create a custom experience.
2. Store Apps
If you cant code there is some built in apps that you can get in the store.
3. Drag-and-Drop Windows
All app windows are draggable, meaning you can position them anywhere on the desktop. This allows for easy multitasking and a flexible workspace.
3. Resizing
Drag the right bottom corner to resize the Window.

How It Works

The GoofyOS app consists of a desktop environment with draggable windows. You can create new apps, view the app store, or open the info page. Here's a basic walkthrough:

  1. Create a New App: Enter a name and HTML content, then click the "Create App" button to add the app to your sidebar.
  2. Open an App: Click on an app in your sidebar to open it in its own window. Apps are displayed in an iframe, allowing you to interact with them in the GoofyOS environment.
  3. Fetch More Apps: Open the store window to fetch more apps from the GitHub repository and add them to your sidebar.
  4. Drag-and-Drop: Move the app windows around to organize your workspace as you like.
